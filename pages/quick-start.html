üîß Backend Setup (Spring Boot)

1. Clone Repository

git clone https://github.com/your-org/healthai-backend.git
cd healthai-backend

2. Configure Database

Edit src/main/resources/application.yml:

spring:
  datasource:
    url: jdbc:postgresql://localhost:5432/healthai_db
    username: postgres
    password: your_password
  jpa:
    hibernate:
      ddl-auto: update
    show-sql: true

  redis:
    host: localhost
    port: 6379

openai:
  api:
    key: your-openai-api-key

vnpay:
  tmnCode: your-tmn-code
  hashSecret: your-hash-secret

3. Install Dependencies

mvn clean install

4. Run Services

# Run Auth Service
cd auth-service
mvn spring-boot:run

# Run User Service
cd ../user-service
mvn spring-boot:run

# Run AI Service
cd ../ai-service
python app.py
